# Realtime Chat Application

A real-time chat application built using Node.js, Express, and Socket.io.

## Features

- Real-time messaging
- Online user counter
- Typing indicator
- Join notifications
- Chat bubbles
- Timestamp for messages
- Dark mode
- Responsive modern UI

## Tech Stack

Frontend:
HTML
CSS
JavaScript

Backend:
Node.js
Express.js
Socket.io

1. Install dependencies

npm install express socket.io cors

2. Start server

node server.js

3. Open frontend

frontend/index.html
